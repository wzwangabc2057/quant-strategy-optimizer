## ⚠️ 缺失/需要处理的数据

### 1. announce_date (公告日期)
- **状态**: ❌ ClickHouse 中不存在
- **当前方案**: 使用 `report_date + 45天` 作为 asof_date
- **风险**: 可能与实际公告日期有偏差（通常在±15天内）
- **建议**:
  - 如需精确，可从 akshare 补充（需要 Python ≥ 3.8）
  - 当前方案在审计中标记为"模拟"

### 2. 退市股票数据
- **状态**: ⚠️ 部分可用
- **说明**: ClickHouse 保留了历史数据，但无法获取完整退市名单
- **影响**: 回测中退市股票会被自动剔除（因为后续无数据）
- **建议**: 当前 Universe 机制已能正确处理

### 3. 上市日期 (list_date)
- **状态**: ⚠️ stockname 表无此字段
- **当前方案**: 通过股票首次出现日期估算
- **风险**: 误差约±5个交易日
- **影响**: 对 MIN_LIST_DAYS=60 的过滤影响较小

### 4. 行业分类
- **状态**: ✅ 已有 stock_block_em
- **可用字段**:
  - `block_type = 'industry'` (行业)
  - `block_name` (行业名称，如"软件开发"、"医疗器械"等)
  - `stock_code` (股票代码)
